  Advanced Hospital Data Visualization  
Interactive insights built using Tableau and cleaned hospital data to support healthcare decision-making.


 Project Overview  
This project focuses on visualizing hospital information using a cleaned dataset (`HospInfo_cleaned.csv`).  
The goal is to help stakeholders analyze:

- Hospital availability & capacity  
- Facility types and service offerings  
- Geographic distribution of hospitals  
- Comparative performance insights  
- Key metrics for strategic planning  

The project includes an interactive Tableau dashboard and a cleaned dataset ready for further analysis.

Key Features

 ✔ Data Cleaning & Preparation  
- Removed null/duplicate values  
- Standardized hospital names and categories  
- Ensured consistent formatting for city, province, and service attributes  
- Verified location data for mapping

 ✔ Advanced Tableau Dashboard (TWBX Provided)  
The dashboard includes multiple insight layers:

- Geospatial maps showing hospital distribution  
- Bar/line charts for hospital type counts  
- Service-category drill-downs  
- Interactive filters allowing user exploration  
- KPIs for total hospitals, facilities, and availability
